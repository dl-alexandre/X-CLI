Features

Everything You Need in Your Terminal

Read, write, and automate X with a native Go CLI designed for speed, security, and AI workflows.

OAuth 2.0 PKCE

Secure authentication with keychain storage. No API keys or client secrets stored in code.

Native Go Performance

200–500ms operations vs 3–5s with browser automation. 6× less memory usage.

Thread Posting

Post long-form content as numbered threads with smart word-boundary splitting and reply chaining.

AI-Friendly Outputs

JSON, Markdown, and table formats. Pipe directly into LLMs with -j or -m flags.

Rate Limit Intelligence

Auto-detection of 429 responses with real-time countdown, automatic retry, and configurable jitter.

Multi-Profile Support

Manage multiple X accounts with named profiles. Switch seamlessly between personal and work.

Integrations

Works With Your Workflow

Pipe to LLMs, integrate with Claude Desktop, or automate with cron.

MCP Integration

Use X-CLI as a tool in Claude Desktop or OpenCode. 8 tools available for reading feeds, searching, and analyzing users.

{
  "mcpServers": {
    "x-cli": {
      "command": "x",
      "args": ["user-summary", "-j"]
    }
  }
}

LLM Pipelines

JSON and Markdown outputs designed for direct piping into AI models. Analyze posts, summarize users, and monitor topics.

$ x search "AI news" -j \
    | llm "Summarize top stories"

$ x user-summary @user -j \
    | llm "Analyze interests"
